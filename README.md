# 🚀 Linguagem de Programação I (Fatec SJC)

Este repositório contém as atividades práticas e exercícios desenvolvidos durante a disciplina de **Linguagem de Programação I** no curso de Banco de Dados da Fatec São José dos Campos. O objetivo principal é consolidar os fundamentos da linguagem Java e os conceitos de Programação Orientada a Objetos (POO).

## 🛠️ Tecnologias Utilizadas
* **Linguagem:** Java 24
* **IDE:** IntelliJ IDEA
* **Gerenciador de Dependências:** Maven 

## 📂 Estrutura do Repositório

O projeto é dividido em módulos que representam cada lista ou conjunto de exercícios:

* **`lp1_ex1` a `lp1_ex3`**: Introdução à sintaxe Java e comandos de saída de dados.
* **`lp1_ex4`**: Implementação de classes para controle de balanço financeiro trimestral e cálculos de média.
* **`lp1_ex5`**: Sistema de cálculo de notas escolares (incluindo lógica para P1, E1, E2, API e Exame Final).
* **`lp1_ex6`**: Primeiros contatos com pacotes e representação de objetos do mundo real (Concessionária e Farmácia).
* **`lp1_ex7`**: Exercícios avançados de POO explorando interações entre objetos:
    * **Feira**: Simulação de compra e venda entre `Cliente` e `Feirante` com controle de estoque e saldo usando `BigDecimal`.
    * **Farmácia**: Sistema de venda de remédios com validação de receita médica e controle de temperatura (Ar Condicionado).
    * **Cabeleireiro**: Gestão de serviços e produtos em um salão.

## 🚀 Como Executar
1. Clone o repositório:
   ```bash
   git clone https://github.com/seu-usuario/lp1_java.git
   ```
2. Abra o projeto na sua IDE de preferência (recomendado IntelliJ IDEA).
3. Navegue até a pasta do exercício desejado e execute a classe `Main.java`.
